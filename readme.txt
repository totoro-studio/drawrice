“抽米v1.0-内测版”抽奖软件
产品特点：
1.本软件为公司企业组织机构各类抽奖活动定制，支持各种形式的抽奖活动；
1.技术上采用最新的Python3 + Qt5 + HTML5 + CSS3，支持最新流行的软件特性；
2.采用本地数据库SQLlite，可无缝对接企业员工数据库，无需另行导入导出；
3.前台采用HTML5 + CSS3，在本软件开放的API框架下，可独立自行定制，HTML5 + CSS3使前台展示界面具有无限可能；
4.逻辑层采用Python3 + Qt5，可以跨平台编译与运行，而不需要修改源代码，并表现平台特有的图形界面风格。底层的C++运行库使本软件具有很高的执行效率；
5.本软件采用RANDOM.ORG提供的最权威随机算法，摈弃编程语言自带的伪随机数算法，保证每次生成随机数的公平公正；
6.操作简易，设定抽奖序列后，一键完成所有抽奖流程，Simple is the Best；
7.所有相关系统参数直接在XML文件中进行配置，保持软件界面的整洁。XML文件可按自已的需求自行设定标签格式，实现丰富多样的软件配置。
8.支持数据导入、保存、查询、导出；
9.支持系统日志和操作日志的双日志功能，使系统运作透明化；
10.前台界面模块化，后期开放软件的界面模块下载功能；
